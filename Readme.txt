https://etf-portfolio.shinyapps.io/Rshiny?

This is a web based ETFs Portfolio construction.

In current version, the ETFs is selected from Yahoo Finacial website by applying MorningStar Box. In next version, we can allow customer to select ETFs. 

In that webpage, the first step is to click the Compute button. After that, you will see the data table of weighted ETFs. 
Then, you can select the Portfolio plot from the combobox of plot. 

Passed the first two steps, you can adjust anything to reconstruct the portfolio.

Note: The Data range is from 2005-01-01 to 2014-12-31, if you want to change the range, Please download ETFs again!!!
